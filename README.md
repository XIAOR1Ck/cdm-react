# CDM-React

CDM stats website in `React + Tailwind`

## Features

- Get Schedule for every seasons (with updated data)
- Get Player Ranking for Seasons and Game Modes

# Planned Features

- Get data of individual players
- Get team data
- Get Map data
- Vods for matches
